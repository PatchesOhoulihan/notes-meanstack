Installing the express cli

To Install use npm install -g express-generator

An Express Project is installed from the command line, and the configuration is passed in using parameters on the command that you use.

When creating a n Express project, you can specify the following options:

  • Which CSS preprocessor to use --css less|stylus

  • Which HTML template engine to use express -ejs (Changes the HTML template engine from jade to EJS) alternative you can use --jshtml or --hogan

  • Whether to add support for sessions

example: express --css less --hogan
